## 🚀 How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/ThanhNguyn/OOP.git
   cd OOP
   ```
2. Open the project in your IDE (VS Code recommended).
3. Run files by language:
   - **C++ (PowerShell):**
     ```bash
     g++ -std=c++23 Week2/Ex1.2.cpp -o ex12
     .\\ex12.exe
     ```
   - **Python:**
     ```bash
     python Week2/Ex2.2.py
     ```
